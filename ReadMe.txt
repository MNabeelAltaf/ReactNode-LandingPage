>> Introduction 

This project is a landing page related to hotel website with navigation and fully responsive design and routing.

> domainname/about or clicking "Home" option of Navbar 
shows landing page of about

> domainname/booking or clicking on "Book Your Stay" in Navbar 
shows Booking screen with fully functional form.

> domainname/404 shows 404 page


>> How to start project locally
For frontend use 
npm start

For backend use
npm run dev


>> Using Seed Data
For using dummy data of hotels and staff, uncomment "seedData()" function 
in index.js file
